# 智能停车场管理系统 python1064

## 项目概述

智能停车场管理系统是基于Python开发的一套适用于各类停车场的在线管理平台。系统实现了一套完整的车位管理流程，包含了前台用户操作与后台管理功能。

## 技术栈

- 开发语言：Python 3.9+
- 数据库：MySQL 5.7
- 前端技术：HTML, jQuery
- 开发工具：PyCharm
- 数据库可视化工具：Navicat

## 功能模块

### 前台系统

- **首页**：展示车位信息概览。
- **车位信息**：显示实时车位状态。
- **车位租用**：提供用户在线租用车位。
- **公告管理**：发布和管理停车场公告。
- **个人中心**：用户查看和管理个人信息。
  - **订单管理**：管理用户的租用车位订单。
  - **租用模拟支付**：模拟支付流程。

### 后台系统

- **个人中心管理**：管理个人信息。
- **用户管理**：对注册用户进行管理。
- **车位区域管理**：设置和调整车位区域。
- **车位信息管理**：维护车位信息。
- **车位租用管理**：处理车位租用相关事务。
- **车位退组管理**：管理用户退租请求。
- **系统管理**：
  - **轮播图管理**：更新首页轮播图。
  - **公告管理**：发布和编辑停车场公告。

## 系统角色

- 系统管理员：负责后台系统管理与维护。
- 普通用户：使用前台系统进行车位租用等操作。

## 运行环境

- 开发环境：Windows
- 架构模式：B/S（浏览器/服务器）

## 部署步骤

1. 环境准备：确保Python、MySQL、Navicat等必备软件安装正确。
2. 数据库搭建：根据设计创建数据库及表结构。
3. 代码部署：将源代码部署到服务器。
4. 系统配置：根据需求配置系统参数。
